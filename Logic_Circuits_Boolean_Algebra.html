✏️ Part 1 — Drawing Logic Circuits from Boolean Expressions

Example: F = A · B + C̄

Step 1: Identify the final operation (lowest precedence) — here it is OR (+)

Step 2: Work inwards: A·B needs an AND gate; C̄ needs a NOT gate

Step 3: Connect outputs of AND and NOT into the OR gate to produce F

A ─┬─── AND ────┐
B ─┘            ├─── OR ──→ F
C ──── NOT ─────┘
⚠️ Operator precedence NOT first, then AND (·), then OR (+). Brackets override this order. 2024 paper feedback: many students ORed B+C before ANDing with D — wrong!
✏️ Student task (3 min) Draw the logic circuit for: F = (A + B) · C̄
Draw it in your notes. Label each gate. What are the inputs to the AND gate at the end?

∧ Part 2 — Boolean Algebra Laws You Must Know

LawAND formOR form
IdentityA · 1 = AA + 0 = A
NullA · 0 = 0A + 1 = 1

💡 De Morgan's — "Break the bar, change the operator"

To negate a whole expression: break the overline into separate NOTs, and switch · to + (or + to ·).

⏱ Part 3 — D-Type Flip-Flop

A D-type flip-flop is a 1-bit memory element — it stores one binary value.

Inputs & Outputs

  • D (Data input)
  • CLK (Clock input)
  • Q (Output)
  • Q̄ (Inverted output — always opposite of Q)

Rules of operation

  1. On a rising clock edge: Q takes the value of D
  2. Between clock pulses: Q holds its current value (memory!)
  3. Q̄ is always the complement (opposite) of Q
⚠️ Common mistake The flip-flop does NOT always toggle. It only changes if D has changed AND there is a rising clock edge.

Tracing the Output — Worked Example

TimeCLK edgeD valueQ (output)Q̄
T1Rising ↑D = 1Q = 1Q̄ = 0
T2No edgeD = 0Q = 1 (held)Q̄ = 0
T3Rising ↑D = 0Q = 0Q̄ = 1
T4Rising ↑D = 1Q = 1Q̄ = 0
T5No edgeD = 1Q = 1 (held)Q̄ = 0
KEY RULE Q only updates on a RISING clock edge. Between pulses, Q holds its last value. This is what makes it a memory element.

➕ Part 4a — Half Adder

A half adder adds two 1-bit numbers (A and B) and produces a Sum and a Carry.

Boolean Expressions

Sum = A ⊕ B   (XOR gate)

Carry = A · B   (AND gate)

2 gates total: one XOR + one AND

Circuit structure

A ─┬─ XOR ──→ Sum
B ─┤
   └─ AND ──→ Carry

Truth Table

ABSum (A⊕B)Carry (A·B)
0000
0110
1010
1101
⚠️ Limitation The half adder CANNOT handle a carry-in. That is why we need a full adder for multi-bit addition.

➕➕ Part 4b — Full Adder

A full adder adds three bits: A, B, and Carry-in (Cin). It is built from two half adders + one OR gate.

Boolean Expressions

Sum = A ⊕ B ⊕ Cin

Cout = (A·B) + (Cin·(A⊕B))

Gates: 2× XOR, 2× AND, 1× OR (5 gates total)

Structure

A, B → [Half Adder 1] → S1, C1
S1, Cin → [Half Adder 2] → Sum, C2
C1, C2 → [OR gate] → Cout

Truth Table (key rows)

ABCinSumCout
00000
01010
10101
11001
11111

Worked Trace: A=1, B=0, Cin=1

1
Half Adder 1: A=1, B=0
S1 = A⊕B = 1⊕0 = 1    C1 = A·B = 1·0 = 0
2
Half Adder 2: S1=1, Cin=1
Sum = S1⊕Cin = 1⊕1 = 0    C2 = S1·Cin = 1·1 = 1
3
OR gate: C1=0, C2=1
Cout = C1 + C2 = 0 + 1 = 1
4
Final result: Sum = 0, Cout = 1
In binary: 1 + 0 + 1 = 10 (decimal 2) ✓

✏️ Task (3 min): Trace for A=1, B=1, Cin=1

Work through: 1) Calculate S1 and C1 from half adder 1   2) Calculate Sum and C2 from half adder 2   3) Calculate Cout from the OR gate

Answer: S1=0, C1=1, Sum=1, C2=0, Cout=1   →   Sum=1, Cout=1 (binary 11 = decimal 3)

📝 Part 5 — Exam Practice Questions

Q1 2 marks

Draw a logic circuit for: F = (A · B) + (Ā · C). Label all gates.

Examiner tipOperator precedence: NOT first, then AND, then OR. Label each gate clearly.
Q2 4 marks

Simplify: F = A·B·C + A·B·C̄ + A·B̄·C + A·B̄·C̄. Show all steps and name each law used.

AnswerFactor A → A·(B·C + B·C̄ + B̄·C + B̄·C̄) → A·(B(C+C̄)+B̄(C+C̄)) → A·(B·1+B̄·1) → A·(B+B̄) → A·1 = A
Q3 2 marks

A D-type flip-flop has Q=0. The clock fires twice: D=1 then D=1. State Q and Q̄ after each pulse.

AnswerPulse 1: D=1 on rising edge → Q=1, Q̄=0. Pulse 2: D=1 on rising edge → Q=1, Q̄=0 (no change, D hasn't changed)
Q4 3 marks

Trace the full adder for A=1, B=1, Cin=0. Show intermediate values S1, C1, C2, Sum, Cout.

AnswerS1=1⊕1=0, C1=1·1=1, Sum=0⊕0=0, C2=0·0=0, Cout=1+0=1. Result: Sum=0, Cout=1 (binary 10 = decimal 2)

Apply laws step-by-step and always name each law. Show all working for full marks.

⏱ D-Type Flip-Flop

📝 Exam technique Show all working, use correct notation, read operator precedence carefully, trace each half adder separately in full adder questions.
