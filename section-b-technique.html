MASTER THESE

The 5 Core Skills

  • 1

    Validated Input Loop

    Repeat getting input until a condition is met. Read once before the loop; re-read at the end of each iteration.

  • 2

    String Traversal

    Access each character by index; compare adjacent pairs using s[i] and s[i+1]. Loop to length−1 when accessing pairs.

  • 3

    Boolean Flags

    Track whether a property holds. Set True before the loop; set False on the first violation. Classify using the flag after the loop.

  • 4

    Counters & Frequency Arrays

    Count occurrences; use fixed-size arrays of 10 slots for digits 0–9. Iterate over the array to find max frequency.

  • 5

    Classify at the End

    After all processing, use tracked values in if/elif/else to determine output. Never classify mid-loop.

DECODE FIRST

The 4-Step Decoding Method

STEP 1 — INPUT

What does the user enter? How many times? Any validation? What type — integer, string?

STEP 2 — PROCESSING

What rules or calculations apply? Work examples by hand first. Trace through with a real number.

STEP 3 — OUTPUT

What are all the different things that might be printed? List every case — how many branches?

STEP 4 — STRUCTURES

Write a bullet list: which loops, variables, flags, arrays do I need? This bullet list earns AO3 Design marks.

PART 2

The 6 Most Common Mistakes (Mark Scheme Evidence)

MISTAKE 1Using maths to extract digits
Convert to string instead — simpler and safer. Using % 10 gives digits in reverse order.
MISTAKE 2Converting chars back to int unnecessarily
Char comparison works directly with < and >. s.charAt(i) < s.charAt(i+1) compares digit characters correctly without conversion.
MISTAKE 3Off-by-one loop bound
Loop to length−2 (i.e. i < s.length()-1) when accessing i and i+1. Otherwise the last iteration throws an index out-of-bounds error.
MISTAKE 4'All same digits' edge case
If both increasing and decreasing flags stay True → the number is NOT bouncy (e.g. 1111). Check for this explicitly.
MISTAKE 5Not resetting variables between outer loop iterations
Reset flags & counters at the start of each outer loop iteration, not just at program start.
MISTAKE 6Not re-reading input inside the validation loop
The input statement must be inside the validation loop body. If it's outside, the loop will repeat forever on the same invalid input.
Mistake 1 — Maths vs String
// WRONG (digits come out backwards):
int temp = n;
while (temp > 0) {
  int d = temp % 10; temp /= 10;
}
// CORRECT (direct index access):
String s = "" + n;
for (int i = 0; i < s.length(); i++) {
  char d = s.charAt(i);
}
Mistake 3 — Loop Bound
// WRONG — crashes on last iteration:
for (int i=0; i<s.length(); i++) {
  char next = s.charAt(i + 1);
  // i+1 out of bounds on last i!
}
// CORRECT:
for (int i=0; i<s.length()-1; i++) {
  char next = s.charAt(i + 1);
  // always safe
}

2024 — Bouncy Number (12 marks)

Java
int number = 0;
while (number <= 0) {
  Console.write("Enter integer > 0: ");
  number = Integer.parseInt(
      Console.readLine());
}
boolean increasing=true,
        decreasing=true;
int goingUp=0, goingDown=0;
String s = "" + number;
for (int i=0; i<s.length()-1; i++) {
  if (s.charAt(i)<s.charAt(i+1)){
    decreasing=false; goingUp++;
  } else if(s.charAt(i)>s.charAt(i+1)){
    increasing=false; goingDown++;
  }
}
if (!increasing && !decreasing) {
  if (goingUp==goingDown)
    Console.writeLine("Perfectly bouncy");
  else
    Console.writeLine("Bouncy");
} else {
  Console.writeLine("Not bouncy");
}
Python
n = int(input("Enter integer > 0: "))
while n <= 0:
  n = int(input("Enter integer > 0: "))
increasing = True
decreasing = True
going_up = 0
going_down = 0
s = str(n)
for i in range(len(s) - 1):
  if s[i] < s[i+1]:
    decreasing = False
    going_up += 1
  elif s[i] > s[i+1]:
    increasing = False
    going_down += 1
if not increasing and not decreasing:
  if going_up == going_down:
    print("Perfectly bouncy")
  else:
    print("Bouncy")
else:
  print("Not bouncy")

2023 — String Validation (All 4 Rules)

Decode first: loop until valid · boolean flag · for loop + nested for · ASCII sum

Java
String input; boolean valid; int codeTotal;
do {
  codeTotal=0; valid=true;
  Console.write("Enter a string: ");
  input = Console.readLine();
  if (input.length()<5||input.length()>7)
    valid=false;
  else {
    for (int i=0; i<input.length(); i++) {
      if (input.charAt(i)<'A'||
          input.charAt(i)>'Z') valid=false;
      else {
        for (int j=0;j<input.length();j++)
          if (i!=j &&
              input.charAt(i)==input.charAt(j))
            valid=false;
      }
      codeTotal += (int)input.charAt(i);
    }
    if (codeTotal<420||codeTotal>600)
      valid=false;
  }
  if (!valid) Console.writeLine("Not valid");
} while (!valid);
Console.writeLine("Valid");
Python
valid = False
while not valid:
  s = input("Enter a string: ")
  valid = True
  total = 0
  if len(s)<5 or len(s)>7:
    valid = False
  else:
    for i in range(len(s)):
      total += ord(s[i])
      if not ('A'<=s[i]<='Z'):
        valid = False
      for j in range(len(s)):
        if i!=j and s[i]==s[j]:
          valid = False
    if total<420 or total>600:
      valid = False
  if not valid: print("Not valid")
print("Valid")

2021 — nth Harshad Number (Search Loop Pattern)

Java
int n = Integer.parseInt(Console.readLine());
int count = 0, candidate = 1;
while (count != n) {
  String s = "" + candidate;
  int digitSum = 0;
  for (int i=0;i<s.length();i++)
    digitSum += s.charAt(i) - '0';
  if (candidate % digitSum == 0) {
    count++;
    if (count==n)
      Console.writeLine(candidate);
  }
  candidate++;
}
Python
n = int(input("Enter n: "))
count = 0
candidate = 1
while count != n:
  digit_sum = 0
  for ch in str(candidate):
    digit_sum += int(ch)
  if candidate % digit_sum == 0:
    count += 1
    if count == n:
      print(candidate)
  candidate += 1

2020 — Modal Digit (Frequency Array Pattern)

Key insight: only 10 possible digits → use array of 10 counters

Java
Console.write("How many digits? ");
int n = Integer.parseInt(Console.readLine());
int[] freq = new int[10];
for (int i=0; i<n; i++) {
  int d = Integer.parseInt(Console.readLine());
  freq[d]++;
}
int maxF = 0;
for (int i=0;i<10;i++)
  if(freq[i]>maxF) maxF=freq[i];
int atMax = 0;
for (int i=0;i<10;i++)
  if(freq[i]==maxF) atMax++;
if (atMax>1)
  Console.writeLine("Data was multimodal");
else
  Console.writeLine(maxF);
Python
n = int(input("How many digits? "))
freq = [0] * 10  # one slot per digit 0-9
for i in range(n):
  d = int(input("Enter a digit: "))
  freq[d] += 1
max_f = max(freq)
at_max = 0
for i in range(10):
  if freq[i] == max_f:
    at_max += 1
if at_max > 1:
  print("Data was multimodal")
else:
  print(max_f)
