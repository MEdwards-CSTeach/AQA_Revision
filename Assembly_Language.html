Part 1

What is Assembly Language?

Key Facts
  • Low-level language — close to machine code
  • One mnemonic ≈ one machine code instruction
  • Processor-specific — not portable between architectures
  • Translated by an Assembler into machine code
  • Direct access to registers and memory
  • Faster execution and smaller code than a HLL
Example — count down from 5:
MOV R0, #5
loop:
  CMP R0, #0
  BEQ done
  SUB R0, R0, #1
  B   loop
done:
  HALT

The AQA Register Model

The AQA model has general-purpose registers R0–R12, plus dedicated registers (PC, MAR, MBR, CIR, SR).

Part 2

Addressing Modes: Immediate vs Direct

How does the CPU interpret the operand in an instruction?

IMMEDIATE  ·  uses # symbol

The # symbol means: use this value directly.

MOV R0, #25   ; R0 = 25 (the number 25 itself)
ADD R1, R1, #1  ; Add the literal value 1 to R1

Exam tip: always starts with #

DIRECT  ·  register name or bare address

Use the value stored at a register or memory location.

LDR R0, 100         ; R0 = value at address 100
ADD R2, R0, R1     ; R2 = R0 + R1

Exam tip: register name or bare memory address

Part 3

Data Movement Instructions

InstructionMeaningExampleEffect
LDR Rd, <mem>Load from memory into register dLDR R0, 100R0 = value at address 100
STR Rd, <mem>Store register d into memorySTR R1, 200Mem[200] = R1
MOV Rd, <op2>Copy value into register dMOV R2, #10R2 = 10
Key Points to Remember
  • LDR and STR always use a bare memory address (no #) — they always use direct addressing
  • MOV can use #value (immediate) or a register Rn (direct to register)
  • LDR: Memory → Register  ·  STR: Register → Memory

Branch Instructions — used with CMP

How CMP works

CMP performs a subtraction and sets flags in the status register — it does NOT store the result. Branch instructions then test those flags.

Branch Conditions
BEQBranch if equal (result zero)
BNEBranch if not equal
BGTBranch if greater than
BLTBranch if less than
BAlways branch (unconditional)

Part 4

Tracing Assembly Language Programs

How to Complete a Trace Table

  • 1
    Set up columnsOne column per register mentioned + one per memory location used
  • 2
    Write initial valuesFill in any starting values given in the question
  • 3
    Execute line by lineWork through each instruction in order, top to bottom
  • 4
    Handle branchesWhen you hit a CMP + branch: check the condition and jump to the label if true
  • 5
    Update only changed valuesOnly write a new value in a cell when it changes
  • 6
    Stop at HALTWrite the final register/memory values in the last row

Part 5

Writing Assembly Language Programs

Writing Tip — Plan First Pseudocode first → identify which registers hold which values → write instructions → add HALT last → add labels → test with a trace table.
Pattern 1: IF / ELSE
  CMP R0, #10
  BGT bigger
  MOV R1, #0   ; small path
  B   done
bigger:
  MOV R1, #1   ; big path
done:
  HALT
Pattern 2: COUNT LOOP
  MOV R0, #0    ; counter = 0
  MOV R1, #10   ; limit = 10
loop:
  CMP R0, R1
  BEQ done
  ADD R0, R0, #1
  B   loop
done:
  HALT

Exam

Exam Technique & Common Mistakes

❌ MistakeWriting the address instead of the value in LDR
✅ CorrectLDR R0, 100 puts the VALUE at address 100 into R0, not the number 100
❌ MistakeForgetting CMP before a branch
✅ CorrectYou must always CMP first — branch instructions check the result of the last CMP
❌ MistakeConfusing # and no # in the operand
✅ Correct# = use this literal number. No # = use the register or memory value
❌ MistakeNot tracing loops carefully
✅ CorrectKeep a new row in the trace table for every time you execute a line
❌ MistakeDescribing instructions not purpose
✅ Correct"Describe the purpose" = what the whole program does, not line-by-line narration
❌ MistakeForgetting HALT at the end
✅ CorrectAll AQA assembly programs must end with HALT
Summary

Summary & Key Exam Points

Addressing Modes
# = immediate (value itself)  ·  Register name or bare address = direct (value at that location)
LDR / STR
LDR: memory → register  ·  STR: register → memory  ·  Both always use direct addressing
CMP + Branch
CMP sets flags; branch conditions: EQ · NE · GT · LT + unconditional B. Must CMP before branching.
Shifts
LSL×1 = ×2, LSR×1 = ÷2. Used in past papers for multiply/divide algorithms.
Tracing
One row per changed state. Track loops carefully. Stop at HALT.
Purpose Questions
Describe the overall task the program performs — not individual instructions.
Why Assembly?
Faster execution, smaller code, direct hardware access — but harder to write and maintain than a HLL.
